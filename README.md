## ‚ú® Features

### 1. App Startup
- Displays a **15-second motivational intro**
  - Random motivational text
  - Random image
  - Skip button available

---

### 2. Task Management (To-Do)
Users can create and manage tasks with the following features:

- Task details:
  - Title
  - Description
  - Color
  - Priority (Low / Medium / High)
- Task status:
  - Pending
  - Completed
- Task options:
  - Repeating tasks (daily / weekly / monthly within a date range)
  - Reminders (enable / disable)
  - Time tracking

#### Task Completion
- Tasks can be marked as completed
- Time spent can be:
  - Entered manually
  - Automatically recorded using Pomodoro sessions

---

### 3. Academic Planner (Event Management)
Designed for fixed-schedule academic activities such as:
- Classes
- Lectures
- Meetings
- Exams

Event characteristics:
- Start time and end time
- Repeating schedule (e.g. weekly classes)
- Reminder support
- ‚ùå No completion status
- ‚ùå No time tracking

---

### 4. Calendar View
- Displays tasks and events by selected date
- Allows adding tasks or events directly from the calendar
- Date color indicators:
  - High priority task ‚Üí Red
  - Medium priority task ‚Üí Yellow
  - Low priority task ‚Üí Green
  - Event-only day ‚Üí Neutral / Blue

---

### 5. Pomodoro Focus Page
- Clean and distraction-free interface
- Customizable Pomodoro settings:
  - Focus time (default: 25 minutes)
  - Short break (default: 5 minutes)
  - Long break (default: 15 minutes)
- Features:
  - Select a task to focus on
  - Automatically track focus time
  - Mark tasks as completed during focus sessions

---

### 6. Reminder & Notification System
- Reminders can be set for both **tasks and events**
- User can:
  - Set reminder date and time
  - Enable or disable reminders
- System sends notification at reminder time if enabled

---

### 7. Reports & Statistics
Provides productivity insights including:
- Total time spent
- Total tasks
- Completed tasks
- Incomplete tasks
- Task completion statistics
- Focus hours
- Daily focus streak
- Visual charts:
  - Bar chart: tasks per day
  - Pie chart: completed vs incomplete tasks
  - Weekly and monthly summaries
